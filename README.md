## Frontend Directory Structure
| Path | Description |
| :--- | --- |
| . | |
| ├── public/ | public access |
| │   └── css | The front style |
| │   └── js | The front scripts |
| │   └── img | The front images |
| ├── views/  | HBS template |
| ├── services/  | Web services |
| ├── routes/  | Web path |
| ├── models/ | Objects Model Classes |
| ├── .index.js  | APP config |

## Backend Directory Structure
| Path | Description |
